 ⚡ DÉMARRAGE ULTRA-RAPIDE (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════

ÉTAPE 1️⃣  - Configuration Discord (2 min)
────────────────────────────────────────

1. Allez sur: https://discord.com/developers/applications
2. Cliquez "New Application" → Donnez un nom (ex: "RoleBot")
3. Onglet "Bot" → "Add Bot"
4. Copiez votre TOKEN (gardez-le secret !!!)
5. Onglet "Bot" → Scroll down → "Privileged Gateway Intents"
   ✅ Message Content Intent
   ✅ Server Members Intent
   ✅ Guilds
6. Onglet "OAuth2" → "URL Generator"
   - Scopes: bot
   - Permissions: Manage Guild, Manage Roles, View Audit Log, Send Messages
   - Copiez l'URL et collez-la dans votre navigateur
   - Sélectionnez votre serveur Discord → Autorisez

✅ BOT INVITÉ SUR VOTRE SERVEUR


ÉTAPE 2️⃣  - Configuration du Projet (1 min)
────────────────────────────────────────────

1. Dupliquez "config/.env.example" → "config/.env"
2. Ouvrez "config/.env" avec un éditeur de texte
3. Remplacez "ton_token_discord_ici" par votre TOKEN
4. Sauvegardez


ÉTAPE 3️⃣  - Lancer le Bot (2 min)
──────────────────────────

Windows:    Double-cliquez sur "scripts/run.bat"
macOS/Linux: Ouvrez Terminal, puis tapez: bash scripts/run.sh


═══════════════════════════════════════════════════════════════════════════
 🎮 UTILISER LE GESTIONNAIRE
═══════════════════════════════════════════════════════════════════════════

SUR DISCORD:
──────────

1. Exécutez la commande:
   !exporter_json

   Le bot envoie un fichier: roles_configuration.json

2. Téléchargez le fichier JSON


DANS VOTRE NAVIGATEUR:
──────────────────────

1. Ouvrez "web/index.html" (double-cliquez)
2. Cliquez "📥 Importer JSON"
3. Sélectionnez votre fichier "roles_configuration.json"
4. Modifiez vos rôles:
   - Déplacez les rôles avec 🔼 🔽
   - Créez de nouveaux rôles avec ✨ Créer un Rôle
   - Supprimez les nouveaux rôles avec 🗑️
5. Cliquez "📤 Exporter JSON Modifié"
6. Téléchargez le fichier modifié


DE RETOUR SUR DISCORD:
─────────────────────

1. Exécutez la commande:
   !importer_json

2. Attachez le fichier JSON modifié

3. Le bot:
   ✅ Crée les nouveaux rôles
   ✅ Réorganise TOUS les rôles selon votre configuration
   ✅ Confirme l'opération

🎉 C'EST TERMINÉ !

═══════════════════════════════════════════════════════════════════════════
 ❌ DÉPANNAGE
═══════════════════════════════════════════════════════════════════════════

Le bot n'apparaît pas en ligne?
   ✅ Vérifiez le TOKEN dans config/.env
   ✅ Vérifiez les Privileged Intents activés
   ✅ Vérifiez que le bot est invité

"PrivilegedIntentsRequired" erreur?
   ✅ Activez "Message Content Intent" en Dev Portal
   ✅ Redémarrez le bot

L'interface web ne charge pas?
   ✅ Ouvrez web/index.html directement (file://)
   ✅ Essayez un autre navigateur
   ✅ Ouvrez la console (F12) pour voir les erreurs

Fichier JSON invalide?
   ✅ Assurez-vous qu'il vient du bot (!exporter_json)
   ✅ Ne modifiez pas la structure manuellement
   ✅ Utilisez uniquement l'interface web
